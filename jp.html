自己紹介

バイリンガルのデータエンジニアとして、AWS・SQL・Linuxを中心とした技術を活かし、スケーラブルなETLパイプラインの構築・改善に取り組んでいます。 アプリケーション開発とデータ基盤運用の双方で培った経験を活かし、効率的で実運用に強いデータソリューションを提供することを得意としています。

現職では、業務要件に応じたSQL・Glue・Lambdaの改修や自動化を通じて、データ品質向上や処理安定化、運用負荷削減に貢献しています。 また、関連チームと連携しながら課題を分析し、迅速かつ実装ベースの改善を推進しています。 現在は、作業手順の自動化やパイプラインの最適化などに取り組みながら、より効率的で信頼性の高いデータ基盤づくりを目指しています。 キャリア初期にはC#/.NETを用いた業務アプリケーション開発を経験し、その後クラウドベースのデータエンジニアリングへ転身しました。

今後も最新技術を取り入れながら、信頼性・拡張性を兼ね備えたデータパイプラインの開発で、事業価値に直結する成果を生み出していきたいと考えています。

AWS Step Function、Lambda、RDS を活用してAWS データパイプラインを構築し、EV データセットを収集・加工・格納。米国の電気自動車データの分析に活用。

プロジェクト詳細

Weather Data ETL project

Airflow、dbt、PostgreSQL を用いた ETL パイプラインを構築し、生データを格納・整形・集計して分析用モデルを作成。ワークフローは Airflow で自動化・監視し、パイプライン全体は Docker 上でコンテナ化して実行。Superset でダッシュボードを作成し、日次・過去の天気傾向を可視化。

プロジェクト詳細

Data Analysis Projects

Python、Pandas、Matplotlib、Seaborn、Plotly を用いて、株式分析、映画データ分析、売上データ分析の3つのプロジェクトを開発。Yahoo Finance などのAPIからリアルタイムデータを取得し、分析と可視化を通じて実用的なインサイトを抽出。
