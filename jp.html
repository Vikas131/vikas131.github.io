自己紹介

バイリンガルエンジニアとして、信頼性が高く効率的なシステム運用に強い関心を持っております。AWS、SQL、Linuxなどの技術を活用し、課題の解決や業務の自動化に取り組んでまいりました。 アプリケーション開発と運用保守の分野て4年以上の経験を有し、現在はデータエンジニアリングとして運用・改善業務に従事しつつ、開発寄りのデータエンジニアリング業務への移行を進めています。

現職では、SQL を用いたデータ分析、抽出、加工、更新にも 取り組み、業務部門や他システム利用者の課題解決を貢献しています。 C#（.NET）を用いた業務ツールの開発に携わり、徐々にクラウドやデータ基盤の分野へシフトしてきました。 現在は、作業手順の自動化やパイプラインの最適化などに取り組みながら、より効率的で信頼性の高いデータ基盤づくりを目指しています。

個人プロジェクト

US EV Data – AWS ETL Project

AWS Step Function、Lambda、RDS を活用してAWS データパイプラインを構築し、EV データセットを収集・加工・格納。米国の電気自動車データの分析に活用。

プロジェクト詳細

Weather Data ETL project

Airflow、dbt、PostgreSQL を用いた ETL パイプラインを構築し、生データを格納・整形・集計して分析用モデルを作成。ワークフローは Airflow で自動化・監視し、パイプライン全体は Docker 上でコンテナ化して実行。Superset でダッシュボードを作成し、日次・過去の天気傾向を可視化。

プロジェクト詳細

Data Analysis Projects

Python、Pandas、Matplotlib、Seaborn、Plotly を用いて、株式分析、映画データ分析、売上データ分析の3つのプロジェクトを開発。Yahoo Finance などのAPIからリアルタイムデータを取得し、分析と可視化を通じて実用的なインサイトを抽出。
